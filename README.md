# TripleCut Tarot

**AI tarot that cuts, not comforts.**

### The Logic
Most tarot readings provide "emotional value". This project does the opposite. 
It uses **TypeScript** to structure uncertainty and **AI** to deliver cold, logical analysis of your situation.

- **No fluff**: No sugar-coating, just patterns.
- **Speed**: Designed for high-frequency decision auditing.
- **Tech Stack**: Built with TypeScript & Vercel.

### How it works
The "TripleCut" logic divides the reading into three distinct analytical layers:
1. **The Pattern** (What's actually happening)
2. **The Friction** (What you are ignoring)
3. **The Outcome** (The logical conclusion)
